# SNIR Experiments — Step 1 Comparison

This folder contains scenarios, data, and plotting helpers used to compare a FLoRa baseline with progressive SNIR model activation.

## Structure
- `scenarios/`: simulation scripts (baseline and SNIR variants).
- `data/`: CSV files generated by scenarios.
- `plots_helpers/`: utility functions for plotting.
- `plots/`: PNG/PDF outputs produced by plotting scripts.

## Prerequisites
- Python 3.10 or newer.
- Project dependencies installed in editable mode:
  ```bash
  pip install -e .
  ```

## Expected workflow
1. Add simulation scripts in `scenarios/` for each variant (baseline, SNIR-only, SNIR + interference).
2. Run each script to populate `data/` with DER/PDR and SNIR metrics.
3. Centralize matplotlib or seaborn helpers in `plots_helpers/` to harmonize figures.
4. Generate comparison plots in `plots/` (PNG and PDF) for step 1 reporting.

## Checkpoints
- Keep the same seed and traffic parameters between baseline and SNIR variants for reproducibility.
- Ensure `flora_mode`, `snir_model`, and `interference_model` are documented in each scenario.
- Include exact configuration details in figure legends for clarity.
