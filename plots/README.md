# `plots/`

Ce dossier contient des **scripts de tracé transverses** hors du pipeline standard `mobilesfrdth plots`.

## Statut

- **Secondaire / support**.
- Utilisé surtout pour des analyses ciblées ou des diagnostics.

## Quand l’ouvrir ?

Quand un workflow avancé vous demande explicitement un script de visualisation spécifique de ce dossier.

Sinon, utilisez d’abord `mobilesfrdth plots` depuis le flux standard documenté dans `docs/user_guide_cli.md`.
