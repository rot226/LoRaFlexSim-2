# `pretest_campagne/`

Ce dossier est la **racine canonique des campagnes de recherche, reproductions métier et archives scientifiques** du dépôt.

## Sous-espaces principaux

- `scenario_a/`, `scenario_b/`, `scenario_c/`, `scenario_d/` : scénarios documentés avec README locaux ;
- `iwcmc_archive/` : archives IWCMC ;
- `archive_or_mock/` : contenus explicitement archivés, dont `mobile-sfrd/` ;
- `common/` : utilitaires partagés entre scénarios.

## Statut

- **Recherche / reproduction / archive**.
- Ce dossier n’est pas le point d’entrée standard pour un nouveau flux utilisateur.

## Parcours conseillé

Commencez par `docs/archive_or_research/README.md`, puis ouvrez le README local du scénario ou de l’archive concerné.
