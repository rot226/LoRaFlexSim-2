## Detailed documentation

### Objectives

- Reproduce required simulations for MNE3SD scenario A.
- Centralize scenario definitions and shared reusable utilities.
- Collect per-scenario metrics as CSV and post-process them into publication-ready figures.

### Shared simulation parameters

Scenario scripts expose a consistent command-line option set:

- `--config`: path to an optional simulation config file overriding defaults.
- `--seed`: base random seed for the simulator.
- `--runs`: number of Monte Carlo repetitions per scenario configuration.
- `--duration`: simulation duration in seconds.
- `--output`: destination CSV file (by convention under `results/mne3sd/scenario_a/`).

Plot scripts in `plots/` follow the same logic:

- `--input`: one or more CSV files generated by scenario scripts.
- `--figures-dir`: output folder for generated figures (default: `figures/mne3sd/scenario_a/`).
- `--format`: export format (`png`, `pdf`, `svg`, etc.).

### Execution profiles

All scenario launchers accept `--profile` (or `MNE3SD_PROFILE`) to switch presets:

- `full` *(default)*
- `fast`
- `ci`

### Generate simulation data

```powershell
python -m pretest_campagne.scenario_a.scenarios.<scenario_module> `
    --runs 10 `
    --duration 3600 `
    --seed 42 `
    --output results/mne3sd/scenario_a/<scenario_name>.csv
```

### Generate figures

```powershell
python -m pretest_campagne.scenario_a.plots.<figure_module> `
    --input results/mne3sd/scenario_a/<scenario_name>.csv `
    --figures-dir figures/mne3sd/scenario_a/ `
    --format pdf
```
